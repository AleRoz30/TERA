# TERA MVP

TERA — навигационный интерфейс к распределённому знанию.  
Он не рекомендует, не оценивает и не ведёт пользователя.

## Что это
- Карта для различения и ориентации в сложном поле информации.
- Не поисковик, не курс, не рекомендательная система.
- Смысл и маршруты задаёт пользователь.

## Быстрый старт
1. Открой приложение.
2. Создай карту или выбери шаблон.
3. Добавляй узлы, связи и источники.
4. Переключай масштаб: 4 → 12 → 36 (слои: state / info / action).
5. Экспортируй карту в `.json` для обмена.

## Принципы
- Нет «правильных» карт и «лучших» путей.
- Нет рейтингов, приоритетов и автосвязей.
- Классификация — это координаты, не интеллект.
- Ответственность за структуру несёт пользователь.

## Данные
- Карты хранятся локально в браузере (IndexedDB).
- Экспорт / импорт `.json` — основной способ передачи.
- Система не использует серверные базы данных.

## Ограничение
TERA не ускоряет понимание.  
Она возвращает ответственность за выбор и интерпретацию.
